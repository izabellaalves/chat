
# 💬 Chat em tempo real

Chat simples usando NodeJS e a biblioteca Socket.IO, esta biblioteca permite comunicação bidirecional em tempo real entre cliente e servidor.



## 📚 Stack utilizada

- Node JS
- HTML
- CSS
- Socket IO
- Express
